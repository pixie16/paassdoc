Detailed Description

A class to output histograms in a format which HHIRF damm is able to read.

This is a special hacked version of the .his file handler which has all of the root dependencies removed. This version is not supported directly. In order to get the most recent version of this file, clone the git repo via git clone https://github.com/cthornsb/his2root.git CRT
