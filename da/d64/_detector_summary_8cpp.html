Override the < operator.

To use the detector summary in the STL map it is necessary to define the behavior of the "<" operator. The "<" operator between a and b is defined as whether the name for a is less than the name for b.

Parameters
[in]a: the Summary on the LHS
[in]b: the summary on the RHS
Returns
true if LHS is less than RHS
