extract channel information from raw data

ReadBuffData extracts channel information from the raw data array and place it into a ChanEvent structure . A pointer to each of the ChanEvent objects is placed in the eventlist vector for later sorting.

Parameters
[in]buf: the buffer to process
[in]bufLen: the length of the buffer
[in]eventList: the event list to add the extracted buffer to
